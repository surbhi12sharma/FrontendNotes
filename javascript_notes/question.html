What is the difference between JavaScript and Java?

Java is an object-oriented programming (OOP) language, while JavaScript is a scripting language. Java is compiled, whereas JavaScript is interpreted.

What are the different data types in JavaScript?

JavaScript has six primitive data types:

  • string
  • number
  • boolean
  • null
  • undefined
  • symbol

Additionally, there is a complex type:

  • object

What is null and undefined in JavaScript?

null is an assignment value indicating that a variable intentionally has no value. It is used to explicitly indicate the absence of any object value.

undefined means a variable has been declared but has not yet been assigned a value. It is the default value of uninitialized variables.
