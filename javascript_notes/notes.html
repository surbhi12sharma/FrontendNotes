JavaScript Notes

1. Variables and Data Types

Definition:

JavaScript is dynamically typed. Variables can be declared using var, let, or const. Common data types include Number, String, Boolean, Object, Array, undefined, null.

Code Snippet:

// Variable declarations
var age = 30;
let salary = 5000.50;
const name = "John Doe";

// Data types
let isEmployed = true;  // Boolean
let hobbies = ["Reading", "Traveling"];  // Array
let person = { name: "Alice", age: 25 };  // Object
let unknown = undefined;  // Undefined
let nothing = null;  // Null

2. Functions

Definition:

Functions can be declared using the function keyword, as function expressions, or as arrow functions.

Code Snippet:

// Function declaration
function greet(name) {
    return "Hello, " + name;
}

// Function expression
const greet = function(name) {
    return "Hello, " + name;
};

// Arrow function
const greet = (name) => "Hello, " + name;

console.log(greet("Alice"));  // Output: Hello, Alice

3. Objects and Prototypes

Definition:

Objects in JavaScript are key-value pairs. Prototypal inheritance allows objects to inherit properties and methods from other objects.

Code Snippet:

// Creating an object
let person = {
    name: "Alice",
    age: 25,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};

// Accessing object properties
console.log(person.name);  // Output: Alice
person.greet();  // Output: Hello, Alice

// Prototypal inheritance
let student = Object.create(person);
student.grade = "A";
console.log(student.name);  // Output: Alice (inherited from person)
console.log(student.grade);  // Output: A

4. Asynchronous JavaScript

Definition:

JavaScript handles asynchronous operations using callbacks, promises, and async/await.

Code Snippet:

// Using Promises
function fetchData(url) {
    return new Promise((resolve, reject) => {
        fetch(url)
            .then(response => response.json())
            .then(data => resolve(data))
            .catch(error => reject(error));
    });
}

// Using async/await
async function getData(url) {
    try {
        const data = await fetchData(url);
        console.log(data);
    } catch (error) {
        console.error("Error:", error);
    }
}

getData("https://api.example.com/data");

5. DOM Manipulation

Definition:

The Document Object Model (DOM) represents the structure of a webpage. JavaScript can interact with the DOM to change content, styles, and handle events.

Code Snippet:

// Selecting an element
const title = document.getElementById("title");

// Changing content
title.textContent = "Welcome to JavaScript!";

// Adding an event listener
title.addEventListener("click", () => {
    alert("Title clicked!");
});

// Creating and appending an element
const newElement = document.createElement("p");
newElement.textContent = "This is a new paragraph.";
document.body.appendChild(newElement);
